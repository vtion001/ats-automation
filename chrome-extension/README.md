# ATS Automation Chrome Extension

**Version:** 2.0.0  
**Platform:** Chrome (Manifest V3)

---

## Quick Start

1. **Load Extension**
   - Go to `chrome://extensions/`
   - Enable Developer mode
   - Click "Load unpacked"
   - Select this folder

2. **Configure**
   - Click the ATS icon in Chrome toolbar
   - Click "Open Config"
   - Enter your Salesforce URL
   - Save configuration

3. **Use**
   - Keep CTM open in one tab
   - Keep Salesforce open in another tab
   - Make/receive calls - automation runs automatically

---

## Features

- Call Detection on CTM
- Automatic Salesforce Search
- Real-time Call Transcription
- AI-Powered Analysis
- Markdown Export

---

## Requirements

- Chrome browser (80+)
- Python 3.10+ (for AI server)
- CTM account
- Salesforce account
