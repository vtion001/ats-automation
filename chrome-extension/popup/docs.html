What is ATS Automation?

ATS Automation is a browser extension that automates repetitive tasks for BPO agents working with multiple client systems. It works by monitoring web pages and automatically performing actions based on triggers.

Quick Start
  1. Install the extension - Load the unpacked extension in Chrome
  2. Select your client - Choose which client profile to use (Flyland, Legacy, TBT, etc.)
  3. Log into your systems - The automation works on your existing logged-in sessions
  4. Enable automations - Turn on the specific automations you need

How It Works

1. DOM Monitoring

The extension monitors web pages for specific elements and events. For example, when a call comes via CTM, it detects the caller information from the page.

2. Automation Triggers

When specific events occur (incoming call, form submission, etc.), automations are triggered to perform actions.

3. Multi-System Integration

Automations can interact with multiple systems simultaneously - for example, looking up a customer in Salesforce while they're on a CTM call.

Security & Privacy
  • No API credentials required - Works on existing logged-in sessions
  • All data stays local - NLP processing happens on your machine
  • No external data transmission - Your data never leaves your computer
  • Transparent operation - You see everything the automation does
Important: Always ensure you are logged into your systems before the automation runs. The automation uses your existing browser session and does not store credentials.
